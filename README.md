#  Rule-Based AI Chatbot

##  Project Overview
This project is a **Rule-Based AI Chatbot** developed using **Python**. The chatbot responds to predefined user inputs using **if-else conditions** and runs continuously until the user exits.

The purpose of this project is to understand the fundamentals of **Artificial Intelligence, control flow, and decision-making logic**.

---

##  Objective
The objective of this project is to create a simple chatbot that:
- Responds to greetings
- Answers predefined questions
- Uses **if-else logic**
- Runs in a continuous loop
- Handles exit commands
